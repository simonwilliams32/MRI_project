
Interpretation of magnetic resonance imaging (MRI) scans is an essential skill for neurosurgeons and neuroradiologists. The ability to decipher between a 'normal' MRI scan and one featuring a pathology is critical. Machine learning can assist with this task.

Here, we introduce MRI_Project - a CNN pipeline able to interpret axial, coronal, and sagittal slices of MRI brain images and classify them as 'normal' or 'abnormal'.
Furthermore, our CNN pipeline is subsequently able to diagnose the specific tumour subtype present on MRI brain images featuring intracranial tumours of class meningioma, pituitary adenoma, and glioma.
